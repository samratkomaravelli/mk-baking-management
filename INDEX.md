## ❓ FAQ

**Q: When should I add a database?**
A: When you want data to persist after page refresh or need to scale to multiple users.

**Q: Can I change the colors?**
A: Yes! Edit `tailwind.config.js` to customize the purple/pink theme.

**Q: How do I add my own contact information?**
A: Edit the contact section in `app/page.tsx`.

**Q: Can I add more cakes?**
A: Yes! Add to the `cakes` array in `app/page.tsx`.

**Q: What's the difference between the old routes and new admin routes?**
A: Old routes: `/orders`, `/expenses`, etc.
New routes: `/admin/orders`, `/admin/expenses`, etc.
You can delete the old routes after migration.

**Q: How do I make updates live?**
A: Commit and push to GitHub, Vercel will auto-deploy (1-2 mins).

**Q: Is the data saved?**
A: Currently stored in React state (resets on refresh). Add a database to persist.
